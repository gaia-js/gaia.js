# 开发Gaia.js

## 全局安装
```
npm i egg-core -g
```

## 配置gaia
```
git clone git@github.com:rockyzh/gaia.js.git

cd gaia.js
ln -s (pwd) /usr/local/lib/node_modules/gaiajs

npm link egg-core
```
